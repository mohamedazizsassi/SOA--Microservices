# TP1 : Tests d'APIs RESTful

Ce dépôt contient 12 fichiers qui testent différentes APIs en utilisant `request`, `fetch` et `axios`.

## APIs Testées
- OpenWeatherMap
- Open Library API
- NASA API
- RandomUser API

## Outils Utilisés
- NodeJS

Chaque fichier correspond à un test d'une API avec une méthode différente.
